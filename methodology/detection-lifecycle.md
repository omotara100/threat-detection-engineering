# Detection Engineering Lifecycle

## 1. Identify Behavior
Understand what malicious or suspicious activity looks like.

## 2. Collect Telemetry
Determine which logs capture the behavior.

## 3. Build Detection
Write detection logic based on reliable indicators.

## 4. Reduce Noise
Account for legitimate use cases.

## 5. Validate
Test detections against known activity.

## 6. Improve
Refine detections based on findings.
